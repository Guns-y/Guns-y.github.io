引用数据类型传递详解

  • 栈内存：保存所有的对象名称 d(保存了引用堆内存空间的地址）

  • 堆内存空间。保存具体对象的具体属性

  • 全局数据区：保存static类型的属性

  • 全局代码区：保存所有的方法定义

    java的引用传递 同一块堆内存空间，同时被多个栈内存指向，不同的栈可以修改同一块堆内存空间的地址
